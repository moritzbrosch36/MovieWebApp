About 🎬 MoviWeb App

MoviWeb App is a small, interactive web application that lets you store, manage, and share your favorite movies.

Features:

  • 👤 Create and select users
  • 🎞️ Add, edit and delete movies
  • 📊 Data is stored in an SQLite database
  • 🖼️ Poster display (optional via API)

Developed with Python, Flask & SQLAlchemy.
